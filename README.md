## 📌 Project Overview

The projects focus on:
- Robotic manipulation  
- Vision-based control  
- Mechanism-based gait generation  

---

# 🔹 Part A: Robotic Arm Pick and Place

## 🧠 Description
This project involves controlling a **Braccio robotic arm** to perform object manipulation tasks using both predefined and vision-based approaches.

## ⚙️ Tasks
- **Pre-Programmed Pick & Place**  
  Transfer object using fixed servo angles (no feedback)

- **Camera-Assisted Pick & Place**  
  Detect object using camera (OpenCV / ArUco) and compute motion using inverse kinematics

- **Object Handover (Two Robots)**  
  Transfer object between two robotic arms using timing coordination

## 🔧 Key Concepts
- Joint-space control  
- Inverse Kinematics (IK)  
- OpenCV & ArUco markers  
- Multi-robot coordination  

---

# 🔹 Part B: Theo Jansen Walking Mechanism

## 🧠 Description
Simulation of a Theo Jansen linkage to generate walking motion using a single rotational input.

## ⚙️ Tasks
- Model linkage mechanism (MATLAB / MuJoCo)  
- Simulate one full motion cycle  
- Track endpoint (foot) trajectory  
- Plot gait pattern in x-y plane  

## 🔧 Key Concepts
- Linkage mechanisms  
- Single DOF systems  
- Gait generation  
- Trajectory analysis  

---

## 📊 Results & Insights
- Successful pick-and-place operations  
- Vision-based system improves flexibility  
- Smooth coordination between two robots  
- Jansen mechanism produces natural walking motion  
- Link adjustments affect stride and step height
